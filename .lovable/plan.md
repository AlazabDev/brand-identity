
## الخطة

### 1. صفحة فريق العمل (`/team`)
- عرض أعضاء الفريق بصور احترافية ومناصبهم وخبراتهم
- تصميم بطاقات أنيقة مع تأثيرات hover

### 2. صفحة مشروع تفصيلية (`/projects/:id`)
- صفحة Case Study لكل مشروع من قاعدة البيانات
- معرض صور، تفاصيل المشروع، المول، المساحة، العميل

### 3. صفحة شركاء النجاح (`/partners`)
- عرض شعارات المولات والعلامات التجارية
- تصميم grid احترافي مع أقسام مقسمة (مولات / علامات تجارية)

### 4. صفحة الوظائف (`/careers`)
- عرض الوظائف المتاحة مع تفاصيل كل وظيفة
- نموذج تقديم متكامل

### 5. تحسين الصفحات الحالية
- إضافة روابط الصفحات الجديدة في Header و Footer
- تحسين التنقل بين الصفحات
